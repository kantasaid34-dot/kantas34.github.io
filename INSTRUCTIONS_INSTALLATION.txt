CALCUL RÉELS — PACKAGE PWA ANDROID (v2) — SAISIE NÉGATIVE CORRIGÉE

NOUVEAUTÉ IMPORTANTE
- Certains claviers Android en mode "decimal" n'affichent pas le signe moins ( - ).
- Cette version ajoute un bouton « ± » à côté du champ de saisie : il permet de mettre/enlever le signe négatif.
- Tu peux donc répondre aux calculs dont le résultat est négatif.

INSTALLATION (RAPIDE)
1) Héberge le dossier sur GitHub Pages (HTTPS) (recommandé)
2) Ouvre l'URL sur Android
3) Menu ⋮ > Installer / Ajouter à l'écran d'accueil

DÉTAILS D'INSTALLATION (PAS À PAS)
Voir les mêmes étapes que précédemment :
- Upload des fichiers dans un dépôt GitHub public
- Settings > Pages > Deploy from a branch > main /(root)
- URL : https://<user>.github.io/<repo>/

REMARQUE CACHE
- Le Service Worker utilise un cache "calculreels-v2".
- Si tu avais déjà installé une ancienne version, il faut parfois :
  * Recharger la page 1-2 fois
  * ou supprimer les données du site (Chrome > Paramètres > Stockage)
